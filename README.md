This project is about creating recipe page for different dishes.

3 different recipes were added to the website. Where main page links to each recipe page separately. There is Turkey Burger,
Taco Casserole, and German Chocolate Dump Cake recipes. 

This project helps to demonstrate basic html knowladge, such as using appropriate headings, list tags, and linking web pages. In addition, it also helps to understand how to search for images, how much to resize them, and how to choose the right alternative text for them.
